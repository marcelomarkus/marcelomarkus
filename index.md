## Projetos Open Source

### [Protocolo ⚡B-Fast](https://pypi.org/project/bfast-py/)
Protocolo binário de alta performance e extrema versatilidade. Projetado para otimizar o tráfego de dados em cenários críticos: desde contextos de LLM (IA) e chats inteligentes até a comunicação ultra-rápida de APIs e sistemas IoT.

### [Bru-gen](https://pypi.org/project/bru-gen/)
CLI que transforma especificações **OpenAPI (Swagger)** em coleções prontas para o **Bruno API Client**. Disponível no PyPI para a comunidade Python.
